## Quick Start

1. Restart resources:
   restart my_vehicles
   restart my_dealership

2. Check console for startup validation
   - Should see: "Server started successfully"
   - Should see: "Loaded 5 vehicles"
   - Should see: "Authorization timeout: 60 seconds"

TEST CASE 1: Normal Purchase Flow
================================================================================

Setup:
1. Join server as player
2. Run: /addmoney cash 100000

Steps:
1. Go to dealership location (-57, -1096, 26)
2. Press E to open dealership UI
3. Purchase "Truffade Adder" ($50,000)
4. Vehicle should spawn at spawn location

Verify:
✓ Money reduced from $100,000 to $50,000
✓ Vehicle spawned with unique plate (8 characters)
✓ Console shows: "Player X purchased Truffade Adder (Plate: XXXXXXXX) for $50000"
✓ Console shows: "VerifyPurchase: Authorized and consumed for plate: XXXXXXXX"
✓ Vehicle persists after restart my_vehicles

Expected Result: PASS

================================================================================
TEST CASE 3: Expired Authorization (Refund Test)
================================================================================

Setup:
1. Edit my_dealership/config.lua
2. Change: Config.AuthorizationTimeout = 5000  (5 seconds instead of 60)
3. Run: restart my_dealership
4. Join server as player
5. Run: /addmoney cash 100000

Steps:
1. Go to dealership location
2. Purchase "Bravado Banshee" ($25,000)
3. IMMEDIATELY press F1 to close NUI (prevent spawn)
4. Wait 10 seconds
5. Check console logs

Verify:
✓ Money initially deducted: $100,000 → $75,000
✓ After 5-6 seconds, console shows: "Refunded $25000 to player X (expired authorization: XXXXXXXX)"
✓ After 5-6 seconds, console shows: "Cleaned up 1 expired authorization(s)"
✓ Money restored: $75,000 → $100,000

Expected Result: PASS

IMPORTANT: Restore Config.AuthorizationTimeout = 60000 after test!

Issue: Refunds not working

Issue: Config validation errors

Fix: AuthorizationTimeout must be number >= 1000

Issue: Plates still duplicating
Check: Is my_vehicles started? (DoesPlateExist export required)
